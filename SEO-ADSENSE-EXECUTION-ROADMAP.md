# ToolsPark SEO + AdSense Execution Kit

## 1) New Site Structure Map

- `/` Homepage
- `/tools` Tools hub
- `/tools/[tool-name]` Tool landing pages (via clean URL rewrites)
- `/blog` Blog hub
- `/blog/[seo-article]` Article pages (via clean URL rewrites)
- `/pages/about`, `/pages/contact`, `/pages/privacy-policy`, `/pages/terms`
- Existing canonical routes remain: `/about`, `/contact`, `/privacy-policy`, `/terms-of-service`

## 2) Homepage SEO Content Blueprint

- **Primary title:** `Free Online Tools for SEO, Calculators and Text Tasks | ToolsPark`
- **Primary meta description:** `Use free online tools for SEO, calculators, text formatting, and productivity. Fast, mobile-friendly, privacy-focused, and no signup required.`
- **Hero H1:** `Free Online Tools That Save You Time`
- **Primary CTA:** `Start Using Free Tools`
- **Above-the-fold priorities:** value proposition, CTA, trust points
- **Below-the-fold priorities:** featured tools, categories, benefits, popular tools, internal links to blog

## 3) SEO Titles + Meta Description Templates

- **Tool page title template:** `[Tool Name] Free Online Tool - Fast, Secure & Easy | ToolsPark`
- **Tool page meta template:** `Use the free [Tool Name] online tool to [primary benefit]. No signup, mobile-friendly, and privacy-focused.`
- **Blog title template:** `How to Use [Tool Name] (Step-by-Step Guide) | ToolsPark`
- **Blog meta template:** `Learn how to use [Tool Name] with examples, tips, and best practices for faster results.`

## 4) Internal Linking Strategy

- Homepage links to: tools hub, top tools, top guides, category sections
- Tools hub links to: every tool page + relevant blog guides
- Tool pages link to:
  - 3-6 related tools
  - 1-3 supporting blog posts
  - tools hub
- Blog pages link to:
  - matching tool page (primary CTA)
  - 2-4 related tool pages
  - 2-3 related articles
- Footer links to legal pages + key tool categories on every page
- Crawl depth target: <= 3 clicks for all indexable pages

## 5) UI/UX Upgrade Checklist

- Consistent global navigation: Home, Tools, Blog, About, Contact + CTA
- Visual hierarchy: clear H1, clear section spacing, readable typography
- Color consistency: high contrast primary action color + neutral background
- Mobile-first spacing and button sizing
- Remove clutter from hero; keep only value + CTA above fold
- Add trust signals: privacy-first, no signup, free usage

## 6) Core Web Vitals Checklist

- **LCP**
  - Compress hero images and use modern formats
  - Preload critical CSS and key fonts only
  - Minimize render-blocking assets
- **CLS**
  - Reserve image/media dimensions
  - Avoid injecting UI above existing content
  - Keep header height stable across breakpoints
- **INP**
  - Reduce heavy JS on load
  - Defer non-critical scripts
  - Avoid long main-thread tasks in tool scripts

## 7) AdSense Readiness Checklist

- Must-have policy pages live and linked in header/footer:
  - About Us
  - Contact
  - Privacy Policy
  - Terms and Conditions
- Original, valuable content:
  - Tool pages include clear explanations and FAQs
  - Blog posts solve user problems with step-by-step value
- Site quality:
  - No broken links
  - Clear navigation and internal linking
  - No thin pages or placeholder pages
- Compliance:
  - Avoid misleading claims
  - Avoid auto-generated spammy copy
  - Keep ad placement policy-compliant (after approval)
