Free Base64 Encoder & Decoder — Encode Any Data Instantly

Base64 is a binary-to-text encoding scheme used widely in web development, APIs, email attachments, and data storage. This free browser-based tool lets you encode any text or file to Base64 and decode Base64 strings back to their original form — with no data ever leaving your browser.

Common uses for Base64 encoding include embedding images directly in CSS or HTML as data URIs, encoding credentials for HTTP Basic Authentication, encoding binary attachments in email (MIME), and passing binary data through JSON APIs.

How to Use the Base64 Encoder

  1. Switch between Text mode (for strings) and File mode (for binary files).
  2. Paste your text or upload your file.
  3. Click Encode or Decode to convert the data.
  4. Copy the output or download the result.

Base64 Encoding FAQ

What is Base64 used for? Base64 is used to safely transmit binary data over text-based protocols like HTTP or email. It encodes binary content as printable ASCII characters.

Does Base64 encrypt data? No. Base64 is encoding, not encryption. It is trivially reversible and should not be used to secure sensitive data.

Is my data safe? Yes. All encoding and decoding happens locally in your browser. No data is sent to any server.
